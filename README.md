# 🐾 Diário Pet

## 📄 Descrição do Projeto
O **Diário Pet** é uma aplicação desenvolvida para ajudar tutores a registrarem e acompanharem o dia a dia de seus animais de estimação.

Com ele, é possível armazenar informações como alimentação, saúde, atividades e observações gerais, facilitando o cuidado e o bem-estar do pet.

---

## 🎯 Objetivo

- Registrar atividades diárias do pet  
- Acompanhar alimentação e rotina  
- Armazenar informações de saúde  
- Organizar dados de forma simples e acessível  

---

## 🚀 Funcionalidades

- 📝 Cadastro de registros diários  
- 🍖 Controle de alimentação  
- 💊 Registro de saúde (vacinas, medicamentos, consultas)  
- 📅 Histórico de atividades  
- 🔍 Visualização de informações do pet  

---

## 🛠️ Tecnologias Utilizadas

- HTML  
- CSS  
- JavaScript  
- Node.js  
- Banco de Dados (SQLite / PostgreSQL)
